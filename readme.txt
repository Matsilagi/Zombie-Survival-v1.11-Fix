-- Install --
Extract, using folder names, to your STEAMACCOUNTNAME/garrysmod/garrysmod folder.

-- Editing the gamemode options --
You can edit the zs_options.lua file for changable values. It's located in garrysmod/gamemodes/zombiesurvival/gamemode/. If you fuck up this file with syntax errors or something then you'll render your installation pretty much useless.
